# Todo Target Dream

## 개요

- Todo, Target, Dream 세 가지 카테고리를 작성합니다.
  - Life 를 위한 Dream
  - Dream 을 위한 Target
  - Target 을 위한 Todo

## 기술

- React-Native
- TypeScript
- Expo
- Firebase (예정)

## 목적

- Todo List CRUD 를 구현합니다.
- Firebase 연결하여 웹, 앱 통합 서비스를 구현합니다.
